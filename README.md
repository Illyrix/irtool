## A tool for translating COBOL to LLVM-IR

### Requirement
* GnuCOBOL >= 2.1.0
* clang >= 3.8.0
* diff >= 3.3
Support Linux only.

### Compile
```
git clone git@github.com:Illyrix/irtool.git
cd irtool
cmake .
make
```

### Usage
use `./irtool -h` or `./irtool --help` to view usage page
